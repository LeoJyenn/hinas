# 海纳思系统管理脚本 (HiNAS Management Tool)

海纳思系统管理脚本是一个功能丰富的Bash脚本，专为海纳思系统设计，提供了丰富的系统管理和配置功能。

## 最近更新

### 2025年新增功能

1. **USB共享文件夹功能**
   - 添加了USB外部设备共享功能
   - 支持创建和删除Samba共享
   - 自动配置共享权限和访问设置
   - 通过Windows网络可直接访问共享文件夹

2. **Nginx站点管理增强**
   - 改进了站点配置列表的显示效果
   - 添加了删除站点配置功能
   - 优化了站点启用/禁用功能的用户体验

## 主要功能

- 常用功能（文件搜索、网络服务重启、系统清理等）
- 中文语言包安装
- 系统检查和状态监控
- Aria2、BT下载工具配置
- 网络测速工具
- 格式化U盘、TF卡
- Docker容器管理
- Cockpit管理界面
- 系统迁移工具
- Tailscale配置
- Socks5服务功能

## 使用方法

海纳思系统中，直接在终端中输入以下命令即可启动：

```bash
caidan
```
